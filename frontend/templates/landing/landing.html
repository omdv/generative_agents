About

Generative Agents are computational software agents that simulate believable human behavior. Each agent has a unique personality, memories, and daily routines. They perceive their environment, remember their experiences, and plan their actions accordingly.

Features

  • Autonomous agent behavior powered by LLMs
  • Memory system with recency, relevance, and importance
  • Natural language conversations between agents
  • Reflection and insight formation
  • Real-time WebSocket updates
